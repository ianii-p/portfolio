## GitHub Pages 업로드 방법 (5분이면 완료!)

### 1단계 — GitHub 계정 만들기
https://github.com 에서 무료 가입

### 2단계 — 새 저장소 만들기
1. 우측 상단 [+] → [New repository]
2. Repository name: `portfolio` (또는 원하는 이름)
3. Public 선택
4. [Create repository] 클릭

### 3단계 — 파일 업로드
1. [uploading an existing file] 클릭
2. index.html 드래그 앤 드롭
3. pdfs 폴더도 드래그 앤 드롭
4. [Commit changes] 클릭

### 4단계 — GitHub Pages 활성화
1. 저장소 상단 [Settings] 탭
2. 좌측 메뉴 [Pages]
3. Branch: main 선택 → [Save]
4. 1~2분 후 URL 생성됨!

### 완성 URL
https://[내 GitHub 아이디].github.io/portfolio/

예: https://yian1118.github.io/portfolio/
